## Persian Voice-Controlled LED System

This project allows you to control LED lights using Persian voice commands. It utilizes Google's Speech Recognition to interpret your voice, processes the command to identify the desired color and state (on/off), and then sends instructions to a NodeMCU device via HTTP requests.

**Key Features:**
*   Real-time Persian voice command recognition.
*   Control of individual LED colors (Red, Yellow, Green) and all LEDs simultaneously.
*   Simple HTTP-based communication with the NodeMCU.
*   Configurable IP address for the NodeMCU.
